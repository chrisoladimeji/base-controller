# base-controller
An ACA-Py controller to derive from for customizations

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```


This controller is developed using Nest.js. For any support please [refer here](https://docs.nestjs.com/support).
